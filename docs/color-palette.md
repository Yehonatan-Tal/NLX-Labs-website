# NLX-Labs Website Color Palette

**Theme: Cool Slate + Living Coral**

| Role | CSS Variable | Value | Usage |
|------|-------------|-------|-------|
| Dark BG | `--color-dark` | `#151A23` | Hero, Process, About, Footer, Nav |
| Dark Surface | `--color-dark-surface` | `#1C2230` | Dark section elevated surfaces |
| Accent | `--color-accent` | `#E0685A` | CTAs, icons, links, glows, dividers |
| Accent Hover | `--color-accent-hover` | `#C85A4E` | Button hover states |
| Accent Soft | `--color-accent-soft` | `#FDF0EE` | Icon badge backgrounds |
| Light BG | `--color-light-bg` | `#F5F6F8` | Services, Industries, Contact sections |
| White | `--color-white` | `#FFFFFF` | Card backgrounds, text on dark |
| Text Primary | `--color-text-primary` | `#1E252F` | Headings, body on light bg |
| Text Secondary | `--color-text-secondary` | `#6B7689` | Descriptions on light bg |
| Dark Text | `--color-dark-text` | `#EDF0F5` | Headings on dark bg |
| Dark Text Secondary | `--color-dark-text-secondary` | `#96A0B4` | Descriptions on dark bg |
| Border Subtle | `--color-border-subtle` | `#E0E3EA` | Light section card borders |
| Border Glass | `--color-border-glass` | `rgba(255,255,255,0.07)` | Dark section card borders |
